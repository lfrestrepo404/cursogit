Comando para inicializar un repositorio

El siguiente comando se utiliza para inicializar un repositorio en Git:

git init

Clonar un repositorio

Para clonar un repositorio remoto, se usa el comando:

git clone https://github.com/usuario/repositorio.git

Ver el estado del repositorio

Este comando muestra el estado actual de los archivos en el repositorio:

git status
